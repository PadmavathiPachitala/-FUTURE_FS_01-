# Personal Portfolio Website

A modern, responsive **Full Stack Developer Portfolio** built with **HTML, CSS, and JavaScript**.
This project showcases my skills, projects, experience, and contact information in a clean and interactive interface.

*LIVE DEMO
https://padmavathipachitala.github.io/-FUTURE_FS_01-/

The website is fully responsive, SEO-optimized, and designed with smooth animations and modern UI principles.

# Features

Modern UI Design

* Clean developer-focused portfolio layout
* Elegant typography using **Syne** and **DM Sans**
* Minimal and professional color palette

Fully Responsive

* Mobile-first design
* Optimized for desktop, tablet, and mobile devices

Interactive Animations

* Scroll reveal animations
* Animated skill bars
* Parallax hero watermark
* Hover effects on project cards

Portfolio Sections

The website contains the following sections:

1. **Hero Section**

   * Introduction
   * Key developer stats
   * Call-to-action buttons

2. **About**

   * Developer background
   * Coding philosophy
   * Technology stack

3. **Resume**

   * Experience
   * Education
   * Certifications

4. **Skills**

   * Technical skill bars
   * Radial skill indicators

5. **Projects**

   * Featured portfolio project
   * CRM system
   * Business website
   * REST API project
   * React weather app

6. **Contact**

   * Contact details
   * Social links
   * Interactive contact form

# Key Functionality

### Scroll Reveal Animations

Elements animate when they enter the viewport using **Intersection Observer API**.

### Animated Skills

Skill bars and radial charts animate when the skills section becomes visible.

### Responsive Navigation

* Desktop navigation
* Mobile hamburger menu
* Smooth scrolling

### Contact Form

Client-side validation included.

Integration options:

* EmailJS (recommended)
* Node.js backend API
